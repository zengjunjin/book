# UI 优化设计方案

**日期**: 2026-06-13
**项目**: 图书推荐系统
**方案**: 极简留白 + 橙色点缀

---

## 设计理念

**极简科技风** — 大量留白 + 清晰的信息层级 + 克制的配色
- 强调内容本身，减少视觉干扰
- 通过字重、大小、颜色区分信息层级
- 橙色仅用于关键交互元素

---

## 配色系统

| 用途 | 颜色 | 说明 |
|------|------|------|
| 页面背景 | `#0f0f14` | 近乎纯黑的深灰 |
| 卡片背景 | `#18181f` | 卡片背景色 |
| 边框 | `#2a2a35` | 细边框分隔 |
| 主色调 | `#f97316` | 橙色，关键元素 |
| 主文字 | `#e4e4e7` | 标题/正文 |
| 次要文字 | `#71717a` | 说明文字 |
| 成功色 | `#10b981` | 成功状态 |
| 危险色 | `#ef4444` | 错误/警告 |

---

## 排版规范

- **标题**: 28-32px，字重 600-700
- **正文**: 15px，行高 1.6
- **说明文字**: 13-14px
- **卡片间距**: 24px
- **页面内边距**: 48px（桌面端）

---

## 组件规范

### 书籍卡片 (BookCard)
- 扁平矩形 + 1px 边框（无阴影/渐变）
- 悬停：边框变橙色 + 背景微微变亮
- 评分显示在右下角，橙色星星
- 预测评分用小标签显示

### 按钮
- **主按钮**: 橙色填充，白色文字
- **次按钮**: 透明背景 + 橙色边框
- **悬停**: 微微缩放（scale 1.02）

### 页面头部
- 大标题 + 副标题（小字灰色）
- 不使用渐变背景，纯色 + 留白
- 搜索框简洁设计

### 侧边栏
- 纯色背景 + 简洁图标
- 选中项有橙色左边框指示

---

## 页面设计要点

### 首页（书籍广场）
- 搜索框居右，标题居左
- 书籍网格使用 4-5 列
- 分页器居中

### 推荐页
- Tab 切换两种算法
- 卡片展示预测评分

### 对比页
- 图表居中展示
- 数据指标清晰对比

### 详情页
- 书籍封面大图展示
- 相似书籍横向滚动

---

1. 更新 App.vue 全局样式（颜色变量）

3. 优化 BookCard 组件（扁平化设计）
4. 优化侧边栏和头部布局
